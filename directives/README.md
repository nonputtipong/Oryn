# Directives

This folder contains **Standard Operating Procedures (SOPs)** written in Markdown.

Each directive defines:
- **Goal** — What the task accomplishes
- **Inputs** — What data or context is needed
- **Tools / Scripts** — Which `execution/` scripts to call
- **Outputs** — Expected deliverables
- **Edge Cases & Learnings** — Known issues and workarounds

Directives are **living documents** — update them as you discover new constraints, better approaches, or edge cases.
